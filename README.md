# AI Resume Analyzer
AI Resume Analyzer is a full-stack web application that helps users analyze their resumes and compare them with job descriptions. It identifies missing skills and generates interview questions using AI.
## Overview
This project works like a simple hiring assistant:
* Upload a resume (PDF)
* Paste a job description
* Analyze resume using AI
* Get skill gap insights and interview questions
## Features
* Resume upload and parsing
* Job description comparison
* Skill gap detection
* Keyword matching
* AI-generated interview questions
## Tech Stack
Frontend:
* React.js
* Tailwind CSS
Backend:
* Node.js
* Express.js
AI:
* LLM API (DeepSeek / OpenAI)
File Handling:
* Multer
* PDF parsing
API Communication:
* Fetch API
## Installation & Setu
git clone https://github.com/abhiraj6386/ai-resume-analyzer.git
cd ai-resume-analyzer
npm install
npm start
## How It Works
1. User uploads a resume
2. Backend extracts text from the file
3. User provides a job description
4. AI analyzes both inputs
5. System returns:
   * Missing skills
   * Suggestions
   * Interview questions
## Use Cases
* Students preparing for placements
* Job seekers improving resumes
* Candidates preparing for interviews
